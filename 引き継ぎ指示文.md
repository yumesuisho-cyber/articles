# 🌙 朝活オラクルカード 新チャット用 指示文

新しいチャットで作業を再開するとき、下の「■ コピペ用」ブロックをそのまま最初のメッセージに貼る。
**毎週変えるのは開始日（○/○）だけ。** 特別な指定がある週は、最後に一文だけ足す（下の「■ 特別指定を足すとき」を参照）。

---

## ■ コピペ用（そのまま貼る）

```
水晶ゆめの朝活オラクルカード記事の続きをお願いします。

まず以下3つを読んで、運用ルールと現状を把握してください（mainブランチにあります）:
1. CLAUDE.md（人格コア・note公開戦略）
2. 「# 🌙 水晶ゆめ 朝活オラクルカード記事 完全運用テンプレート v3.5.md」（記事の型）
3. 「朝活オラクルカード_引き継ぎメモ.md」（決定事項・カード使用履歴・残タスク）

運用ルールの要点:
- カードはSacred Forest Oracle全52枚。同じカードは14日以上あける（引き継ぎメモの使用履歴表で「次回使用可能日」を必ず確認）。未使用・長期未使用を優先して選ぶ。
- ハッシュタグはnote・Xとも同じ5個で固定: #オラクルカード #占い #セルフケア #水晶ゆめ #心をほどく言葉マガジン（必要な日は7個まで任意で追加可）。
- 記事フォーマットはv3.5（スリムタイトル／プロンプトは末尾「おまけ」にコードブロックで①ゆめ②ルナ／イラストセクションに「※Nano Banana 2で生成しました」は入れない）。
- 記事はmainに直接置く方針。作業ブランチ claude/oracle-card-articles-m6Prb は古いので使わない。

今回の依頼: ○/○（月）から1週間分の記事を作成して、mainにコミット＆プッシュしてください。カードは引き継ぎメモの使用履歴で「次回使用可能日」を確認し、未使用・長期未使用を優先して選ぶこと。季節の行事や気候があれば軽くフックに使ってOK。作成後は引き継ぎメモのカード使用履歴に追記してください。
```

---

## ■ 毎週やること

- 「今回の依頼」の **○/○ を、その週の開始日（月曜）に差し替える**だけ。
- それ以外は基本さわらない。

## ■ 特別指定を足すとき（その週だけ）

「今回の依頼」の末尾に一文足す。例：

- 特定カードを必ず使いたい → 「◯◯（テーマ）をこの週のどこかで必ず使うこと。」
- 行事に寄せたい → 「◯/◯は◯◯（例：海の日）なので季節フックに使ってOK。」
- テーマを縛りたい → 「今週は全体を『癒し寄り』でまとめてください。」

## ■ 覚えておくと安心なこと

- 記事は `yume/` フォルダに `2026-MM-DD_カード名.md` の形で入っている。
- カードの選定・14日ルールの管理は、**引き継ぎメモのカード使用履歴表が最新の一次情報**（テンプレ本体の履歴表は古いまま更新していない場合がある）。
- 未使用・優先カードの状況が変わったら、引き継ぎメモ側を更新しておくと次回が楽。
