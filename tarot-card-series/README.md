# 私とルナの22枚タロット工房｜夜空で紡ぐ物語

## シリーズ概要

私（水晶ゆめ）とルナのイラストを使ったオリジナルタロットカードを制作するシリーズ。
大アルカナ22枚を順番に紹介し、読者投票で本採用を決める参加型企画。

### 制作ツール
- **イラスト生成**: ナノバナナプロ（参照画像あり）
- **フレーム付け**: PixAI（チャット編集機能）

### キャラクター使い分け
- **私（ゆめ）**: 意志・行動・感情・人間的側面のカード
- **ルナ**: 神秘・直感・女性性が強いカード、2人の場面

## サムネイル・マガジン画像プロンプト

### note記事サムネイル（ナノバナナプロ用）

```
提供する2枚のキャラクター画像をもとに、2人のキャラクターが並んで登場する
「私とルナの22枚タロット」用サムネイル画像を作成してください。

【テーマ】
夜空に浮かぶタロットの工房。静かで神秘的でありながら、手作りの温かみを感じる創作の場所。

【構図】
横長構図で、中央上部に大きなタイトル用スペースを確保。
左下にキャラ1（私・ゆめ）、右下にキャラ2（ルナ）を約3頭身のゆるデフォルメで配置。
2人は優しく微笑みながら、タロットカードを作っている様子、または完成したカードを嬉しそうに見せているポーズ。

【タイトル（必須）】
私とルナの
タロット工房
※読みやすい太字風・少し丸みのあるポップな日本語フォント風デザイン。
※色は白または薄い金色に縁取りをつけて、夜空の背景に映えるように。
※2行で表示し、人物の顔に被らない中央位置に大きく配置。

【背景】
深い紫から明るい藤色へのグラデーション（夜明け前の空）。
パステルピンクと水色を星の光として点在。
金色の三日月、小さな星、にじんだ光の粒子を散りばめる。

【工房らしい装飾】
背景にうっすらと、以下を透明感のあるタッチで配置：
- 描きかけのタロットカードのシルエット
- 羽ペンやインク瓶
- 作業机や本棚の輪郭
- 水晶の形や光の輪
装飾は控えめに、人物とタイトルを邪魔しない程度に。

【キャラクター】
外見（髪型・服装・色・アクセサリー）は参照画像と完全一致。
約3頭身のゆるデフォルメで、顔は丸く、目は少し大きめで優しい表情。
2人とも創作への喜びと希望に満ちた表情で、一緒にタロットを作っている雰囲気。

【禁止】
- 長文の文章を絵に書かない（タイトルのみ）
- キャラの外見変更は禁止
- 4コマ形式の線を入れない
```
