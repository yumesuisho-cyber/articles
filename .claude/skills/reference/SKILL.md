# 参照画像イラストプロンプト生成スキル（reference・水晶ゆめ仕様）

参照画像を使ってキャラのイラストを描くためのプロンプトを生成する。
設計の考え方はきょん(るびぃ)さんの/referenceスキルに基づく: **見た目(髪色・瞳・服)は参照画像に任せて描写を省き、参照だけでは崩れやすい「体型・プロポーション」「必ず残す特徴」と、場面ごとに変えたい「表情・ポーズ・視線・構図・雰囲気」だけを言葉で書く**。
GPT-Image-2・PixAI Edit Pro・PixAI Reference Pro のいずれでも使えるよう、日本語の自然文プロンプトで出力する。

## 呼び出し方

```
/reference [シチュエーション]
```

- 例: `/reference ゆめとルナが窓辺で朝のオラクルカードを一枚引いている`
- シチュエーション省略時はヒアリングする。

## プリセットキャラクター（ヒアリング不要）

| キャラ | 体型・プロポーション | 必ず残す特徴 |
|---|---|---|
| ゆめ | すらりとした大人の女性。やさしい佇まい | 金の星のブローチ、淡く光る宝石のネックレス |
| ルナ | 小柄な女の子(ゆめより頭ひとつ以上低い) | 猫耳フード・白いふわふわのしっぽ・水晶玉のペンダント |

- 2人が並ぶ場合は身長差(ゆめ>ルナ)を必ずプロンプトに含める。
- プリセット外のキャラが登場する場合のみ、体型・残したい特徴をヒアリングする。

## 実行手順

### STEP 0: ヒアリング（足りない分だけ）

1. **シチュエーション** — 引数が空なら確認(場所・ポーズ・表情・雰囲気)
2. **登場キャラ** — ゆめ/ルナ/両方/その他。複数なら誰を何枚目の参照画像にするかを確認
3. **比率** — 毎回確認(16:9/1:1/9:16など)。note記事用なら16:9をデフォルト提案
4. **声(配色)** — 癒しの記事(ラベンダー×白×金・やわらかい光)か、ネオンの記事(夜背景×ピンク×シアンの発光)かを確認。デフォルトは癒し

### STEP 1: プロンプト生成の原則

- **見た目(髪色・瞳・服・小物)の描写は書かない** — 参照画像に任せる
- 書くのは: 体型・プロポーション(プリセット)/必ず残す特徴(プリセット)/表情・ポーズ・視線・配置/構図・雰囲気
- **画風は参照画像の絵柄・塗りに寄せる**。肌や髪にやわらかいグラデーションとツヤを入れ、フラット塗りにしない。背景も人物と同じ画風で統一(no photorealistic, no 3DCG)
- **文字・ロゴは一切入れない**(末尾で念押し)。文字入りが必要な場合は図解イラスト制作ガイドの文字ルールに従う
- 重み記法 `(xxx:1.4)` は使わない。日本語の自然文で書く
- カメラ目線にしすぎない。複数キャラは視線の向き・関係性(向かい合う/同じものを見る)を書く

### STEP 2: 出力フォーマット

````
🎨 参照イラスト（GPT-Image-2 / PixAI Edit Pro / PixAI Reference Pro 共通 / [比率] / 文字なし / [登場キャラ]）

```
[比率・雰囲気の導入文]

【画風指定】: 参照画像のキャラと同じ絵柄・同じ塗りに寄せる。肌や髪にはやわらかいグラデーションとツヤ・ハイライトを入れ、のっぺりしたフラット塗りにはしない。背景・人物ともに同じ画風で統一し、背景だけ写実的・実写風・3DCG風にしない。no photorealistic, no 3DCG。

[シーン・場所の説明。声に合わせた配色(癒し=ラベンダー×白×金/ネオン=ピンク×シアンの発光)]

[1枚目の参照画像のキャラクター(ゆめ): 表情・ポーズ・視線・配置。すらりとした大人の女性。金の星のブローチと宝石のネックレスは必ず残す]
[2枚目の参照画像のキャラクター(ルナ): 表情・ポーズ・視線・配置。小柄な女の子でゆめより頭ひとつ以上低い。猫耳フード・白いしっぽ・水晶玉のペンダントは必ず残す]

【構図】: [比率・視線・配置・全体ムード]

画像内には文字・文章・単語・ロゴを一切入れないこと。
```
````

## サムネモード（文字入り）

`/reference サムネ [記事名やシチュエーション]` のように「サムネ」と言われたら、文字入りモードに切り替える。

### 追加ヒアリング
1. **入れる文字** — メイン(大きく)とサブ(小さめ)の2段構成を基本に、正確な文言を確認する(例: メイン「当てない占い師×夢を灯す創作家」/サブ「水晶ゆめ｜自己紹介」)
2. 比率は16:9をデフォルト提案(noteの見出し画像用)

### サムネ用の追加ルール(通常ルールとの違い)
- 「文字・ロゴを入れない」念押しは外し、かわりに**文字ブロック**を入れる:
  > 画像内に以下の文字を、誤字なく正確に描き込んでください。指定した文字以外は入れないこと。
  > メイン(中央上部・大きく): 「◯◯」
  > サブ(メインの下・小さめ): 「◯◯」
  > 文字スタイル: やさしい丸みのある日本語フォント風。癒しの声なら白〜淡い金色+柔らかいグロー、ネオンの声なら発光するネオン文字。背景とのコントラストを確保し、スマホのタイムラインでも読める大きさに
- **文字とキャラの顔は中央寄り(上下の端から1割は空ける)** — noteの表示枠(1.91:1)で上下が少し切られるため
- 出力の最後に必ず添える注意書き: 「公開前に文字を目視校正。崩れたら文字なし版を生成してCanva等で後入れに切り替え」

## 注意事項

- 実在デッキ(Sacred Forest Oracle等)のカード絵柄は描かせない。カードは裏向きか抽象的なオリジナル柄にする
- ゆめブランドの絶対ルール: 読者を責める・怖がらせる表現にしない。困り顔は「少し困った」まで
- GPT-Image-2でフィルターに弾かれたら、同じプロンプトをPixAI Edit Pro/Reference Proで試せる
- 生成後に表情・ポーズ・構図の修正要望を聞き、修正指示文(「方向性は良いです。〇〇だけ直してください」形式)を出す
