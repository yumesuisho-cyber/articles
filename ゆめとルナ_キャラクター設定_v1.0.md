# ゆめとルナ｜キャラクター設定 v1.0

*(2026-07-20 キャラ設定シート2枚から転記。創作短編シリーズ・タロット工房・挿絵生成の共通基準にする)*
*(イラスト生成時の固定要素は「.claude/skills/reference」のプリセットと連動。ここが正、変更時は両方更新する)*

## ゆめ（Yume）

- 身長: 160cm
- 役職: 占いライター
- モチーフ: 星・月・タロット
- 性格: おっとり・繊細・共感型
- 内面の設定: 人見知りで話すのは苦手・自己肯定感が低め・自己アピールが苦手。だからこそ「書く言葉」を選んだ（文字の中でなら誰かのそばに座れる）。物語でゆめの迷いを描く時はこの感覚を軸にする

**見た目（固定要素）:**
- 金髪ロング（ゆるいウェーブ・編み込みのハーフアップ）
- 紫の瞳
- 紫のドレス（濃紫のオフショルダー＋淡紫のスカート重ね。月と星の金刺繍）
- 胸元に金の星の飾り、淡く光る宝石のペンダント
- 金のチェーンベルト（小さな壺型の金具）
- シルバーのヒール

**カラーパレット:** 髪=クリーム〜金 / 瞳=薄紫 / ドレス濃色=濃紺紫 / ドレス淡色=薄紫 / 装飾・金具=金 / アクセサリー・靴=淡紫〜白

## ルナ（Luna）

- 身長: 140cm（ゆめより頭ひとつ以上低い）
- 役職: 月の精霊・ゆめの相棒
- モチーフ: 月・星・猫耳
- 性格: 元気いっぱい・好奇心旺盛

**見た目（固定要素）:**
- 銀白色の髪
- 紫の瞳
- 黒の猫耳フード付きマント（金の月と星の柄。フードから白い猫耳）
- グレーのワンピース（月と星のチェーン飾り）
- 白いふわふわのしっぽ
- チョーカー＋水晶玉のペンダント
- 白いファーブーツ

**カラーパレット:** 髪=白銀〜グレー / 瞳=紫 / マント=黒（月星柄） / 服=グレー / 装飾=金＋紫の水晶 / 靴=白

## 2人の関係と役割分担（物語・記事共通）

- 静と動のバディ。ゆめが受け止めて言葉にする側、ルナが動いて世界を広げる側
- アニメ公式設定: ゆめ＝タロットの意味と人間の感情を深く知る「知識の先生」／ルナ＝各カードの世界への扉を開く力を持つ「冒険の案内役」
- ルナはゆめを「ゆめちゃん」と呼ぶ
- タロット工房では担当カードを交互に持つ（恋人など2人のカードは両方登場）
- 並ぶ絵では身長差（ゆめ＞ルナ）を必ず反映する

## 物語上の約束事（幕間シリーズ）

- ルナの由来（なぜゆめのもとへ来たのか）は明かさない。アニメ本編が将来描く可能性があるため、幕間では「月はぜんぶ見てるんだよ」以上を語らせない
- ゆめの手帳「月の栞」: 旅立ちの夜（幕間第0話）から書き始めた旅の記録。幕間各話は「月の栞に挟まれた一夜のページ」

## 運用メモ

- 認知のための固定要素（変えないもの）: ゆめ=金髪・紫の瞳・星のブローチ・宝石のペンダント／ルナ=猫耳フード・白いしっぽ・水晶玉のペンダント
- 衣装替え・季節版・ネオン版などの遊びは可。ただし上記の固定要素は残す
- もっと細かい設定（口調・好きなもの・過去など）は、創作短編シリーズの企画設計時に追記する
