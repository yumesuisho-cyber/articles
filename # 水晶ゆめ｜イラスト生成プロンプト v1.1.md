\# 水晶ゆめ｜イラスト生成プロンプト v1.1  
使用モデル：Nano Banana Pro  
最終更新：2026年3月

\---

\#\# ■ 基本方針

記事の「図解」ではなく、\*\*読者の感情に寄り添う情景イラスト\*\*として機能させる。  
テキストを説明するより、\*\*記事の世界観・空気感を視覚化\*\*することを優先する。

\---

| 項目 | 設定 |  
|---|---|  
| 髪 | 金髪ロングヘア |  
| 瞳 | 紫の瞳 |  
| 衣装 | 紫がかったドレスまたはゆったりしたワンピース |

\#\# ■ スタイル定義

| 記号 | スタイル名 | 特徴 |  
|---|---|---|  
| \*\*A\*\* | 手書き風（デフォルト） | 水彩・色鉛筆タッチ。温かみのあるパステル調。ラフな線画、デジタル感を排除 |  
| \*\*B\*\* | ビジネス風 | フラットデザイン。整ったベクター調。情報の視認性優先 |  
| \*\*C\*\* | ポップ・コミック | 太い主線・鮮やかな原色。元気な印象 |  
| \*\*D\*\* | シンプル・ミニマル | 線画のみ、最低限の色数。洗練された印象 |

\---

\#\# ■ レイアウト定義

| 記号 | レイアウト名 | 向いている場面 |  
|---|---|---|  
| \*\*A\*\* | 並列リスト | 要点まとめ・ポイント列挙 |  
| \*\*B\*\* | 比較図 | VS構造・左右対比 |  
| \*\*C\*\* | ステップ進行 | 手順・ロードマップ |  
| \*\*D\*\* | 4象限マトリクス | 分布・ポジショニング |  
| \*\*E\*\* | サイクル図 | 循環・ループ |  
| \*\*F\*\* | ピラミッド | 階層構造 |  
| \*\*G\*\* | 複合・お任せ（デフォルト） | 内容を解析して最適な構成を自動で組む |
